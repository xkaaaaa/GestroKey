# GestroKey - 隐绘助手

---

## 项目简介

GestroKey 是一个基于手势识别的快捷操作工具，通过绘制特定手势来快速执行各种操作。它可以帮助用户提高工作效率，实现快速、隐蔽的操作控制。

## 主要功能

- 手势识别：支持多种自定义手势识别
- 快捷操作：通过手势快速执行各种操作
- 界面简洁：简单直观的用户界面
- 跨平台支持：基于Python开发，支持多平台运行

## 安装与使用

1. 克隆本项目
   
   ```bash
   git clone https://github.com/yourusername/GestroKey.git
   ```

2. 安装依赖
   
   ```bash
   pip install -r requirements.txt
   ```

3. 运行程序
   
   ```bash
   python src/main.py
   ```
